# Bot Telegram

## 📌 Descripción
Bot Telegram - Es un asistente virtual que cuenta con juegos e historias que interactuan con el usuario.

## 🔧 Tecnologías
- Pyrogram
- Random
- Datetime
- FusionBrainIA
- Base64

## 📝 Funciones 
1. **Funcionamiento a comandos**
2. **Botones interactivos**
3. **Juegos como Piedra, Papel o Tijera, Adivina el Número**
4. **Historia interactiva a base de decisiones**
5. **Precisa la hora y fecha**
6. **Generación de imágenes**
