# Bot Telegram

## 📌 Descripción
Bot Telegram - Es un asistente virtual que cuenta con juegos e historias que interactuan con el usuario.

## 🔧 Tecnologías
- Pyrogram
- Random
- Datetime
- FusionBrainIA
- Base64

## 📝 Funciones 
- Funcionamiento a comandos
- Botones interactivos
- Juegos como Piedra, Papel o Tijera, Adivina el Número
- Historia interactiva a base de decisiones
- Precisa la hora y fecha
- Generación de imágenes
